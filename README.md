# MICP-UNET-Project: Microscopic Dark-field Image Segmentation

### 🔬 Project Overview (项目概览)
This repository focus on image segmentation under **microscopic dark-field environments**. It is specifically developed for **MICP (Microbially Induced Calcite Precipitation)** research to automatically identify **crystals** and **PDMS structures** in microfluidic chips.
(本项目专注于显微镜暗场环境下的图像分割，专为 MICP 研究开发，用于自动识别微流控芯片中的晶体和 PDMS 结构。)

---

### 📂 File Structure & Usage (文件结构与用法说明)

#### 🚀 Core Scripts (核心脚本)
* **`test.py`**: **[Final Version]** This is the ultimate integrated script for final testing and prediction. Use this for your main results.
  (【最终版本】这是最终的集成脚本，用于最终的测试和预测。请以此文件作为主要运行程序。)
* **`train_UNET.py`**: Main program for training the U-Net model.
  (U-Net 模型训练主程序。)
* **`json_label.py`**: Pre-processing tool for handling JSON labels generated by Labelme.
  (用于处理 Labelme 标注生成的 JSON 数据的预处理工具。)

#### 📝 Development History (改进记录)
* **`test_UNET.py` / `test_UNET_File.py`**: These scripts record the incremental improvements and experimental iterations of the code.
  (这些脚本记录了代码改进过程中的中间版本和实验迭代过程。)
* **`draw.py`**: Visualization tool for plotting results.
  (结果可视化与绘图工具。)

#### 📦 Modules (功能模块)
* **`network/`**: Contains the definition of U-Net architecture. (存放网络结构定义。)
* **`utils/`**: Includes data loading and image pre-processing functions. (包含数据加载与预处理函数。)

---

### 🌟 Key Features (核心亮点)
* **Dark-field Optimization**: Optimized for images with high contrast (bright targets on black backgrounds).
  (针对暗场高对比度图像进行了优化，即黑底亮色目标。)
* **Dual-Target Recognition**: Capable of distinguishing geometric crystals from complex PDMS boundaries.
  (能够精准区分规则的晶体几何形状与复杂的 PDMS 边界。)
